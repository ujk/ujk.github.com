Eleman Seçicisi (“element”)

element selector
Açıklama : Verilen tag ismindeki elemanları bulur.

jQuery( "element" ) Eklendiği Versiyon 1.0
element : bulunmak istenen DOM elemanı adı.

Bu seçici kullanılırken jQuery getElementsByTagName() JavaScript metodunu kullanır.



Örnekler:

Tüm <div> elemanlarını bulur.

<style>
   div, span {
      width: 80px; height: 80px;
      float: left; padding: 10px;
      margin: 10px;
      background-color: #eee;
   }
</style>
 
<div>DIV1</div>
<div>DIV2</div>
<span>SPAN</span>
 
<script>
   $( "div" ).css( "border", "9px solid red" );
</script>
DIV1
DIV2
SPAN
