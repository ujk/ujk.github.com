:even Seçicisi

even selector Vazgeçilen versiyon 3.4
Açıklama : Sıfır endeksli olarak çift elemanları bulur. Ayrıca :odd seçicisine bakınız.

jQuery( ":even" ) Eklendiği Versiyon 1.0

JQuery 3.4 itibarıyla :even sahte sınıfından vazgeçildi. Eğer varsa seçicilerinizden onu çıkarıp yerine sonuçları .even() metodu ile filtreleyiniz. (Versiyon 3.5.0 ve sonrası için)

Dikkat ediniz :even seçicisi sıralamaya sıfırdan başladığı için ilk, üçüncü, beşinic vs elemanları seçmektedir.



İlave Notlar :

  • :even bir jQuery eklentisi olduğu ve CSS standardı olmadığı için JavaScript querySelectorAll() metodu performansından yararlanamaz. Modern tarayıcılarda daha iyi bir performans için önce saf CSS seçici ile elemanları seçip .filter(":even"). ile seti filtreleyiniz.
  • Seçilen elemanlar dökümanda yer aldıkları sırayla gelirler.


Örnekler:

Çift sıra numaralı tablo satırlarını bul (index'i 0, 2, 4 olanlar).

<style>
   table { background: #eee; }
</style>
<table border="1">
   <tr><td>Row with Index #0</td></tr>
   <tr><td>Row with Index #1</td></tr>
   <tr><td>Row with Index #2</td></tr>
   <tr><td>Row with Index #3</td></tr>
</table>
 
<script>
   $( "tr:even" ).css( "background-color", "#bbf" );
</script>
Row with Index #0
Row with Index #1
Row with Index #2
Row with Index #3
