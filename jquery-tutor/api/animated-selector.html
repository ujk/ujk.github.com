:animated seçicisi
Açıklama : Seçici çalıştırıldığı anda animasyon yapmakta olan tüm elemanları seçer. ------------------ TO DO ------------------

jQuery( "*" ) Eklendiği Versiyon 1.0

Dikkat : All ya da üniversal seçicisi oldukça yavaş çalışır.


Örnekler:

Dökümandaki tüm elemanları seç (head, body, html dahil). Dikkat edin eğer tarayıcınız DOM elemanlarına <script> ya da link ekleyen eklentilere sahipse o ekledikleri de sayılır.

<style>
   h3 { margin: 0;}
   div, span, p {
      width: 100px;
      height: 80px;
      float: left;
      padding: 10px;
      margin: 10px;
      background-color: #EEEEEE;
   }
</style>
 
<div>DIV</div>
<span>SPAN</span>
<p>P <button>Button</button> </p>
 
<script>
   var elementCount = $( "*" ).css( "border", "3px solid red" ).length;
   $( "body" ).prepend( "<h3>" + elementCount + " eleman bulundu</h3>" );
</script>

Sadece görünen elemanları seçmek için bir <div> içine alabilirsiniz.

<style>
   h3 { margin: 0; }
   div, span, p {
      width: 100px; height: 80px;
      float: left;
      padding: 10px;
      margin: 10px;
      background-color: #EEEEEE;
   }
   #test {
      width: auto;
      height: 120px;
      background-color: transparent;
   }
</style>

<div id="test">
  <div>DIV</div>
  <span>SPAN</span>
  <p>P <button>Button</button></p>
</div>
 
<script>
   var elementCount = $( "#test" ).find( "*" ).css( "border", "3px solid red" ).length;
   $( "body" ).prepend( "<h3>" + elementCount + " eleman bulundu</h3>" );
</script>
