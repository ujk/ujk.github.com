Child Seçicisi (“parent > child”)

child selector
Açıklama : parent ile verilen üst elemanın içindeki child ile belirtilen alt elemanları seçer.

jQuery( "parent > child" ) Eklendiği Versiyon 1.0
parent : herhengi geçerli seçici

child : alt elemanları filtrelemek için seçici

Bu seçici (E > F) diğer alt elemanları bulan seçici (E F) nin özel versiyonudur. $("E F") seçimi E elemanı içindeki tüm F elemanları bulurken $("E > F") sadece birinci derecede direk alt eleman olan F elemanları bulur.



Örnekler:

<ul class="topnav"> içindeki <li> elemanları bul ve çerçece içine koy.

<ul class="topnav">
   <li>Item 1</li>
   <li>Item 2
      <ul>
      <li>Nested item 1</li>
      <li>Nested item 2</li>
      <li>Nested item 3</li>
      </ul>
   </li>
   <li>Item 3</li>
</ul>
 
<script>
$( "ul.topnav > li" ).css( "border", "3px double red" );
</script>
  • Item 1
  • Item 2
    • Nested item 1
    • Nested item 2
    • Nested item 3
  • Item 3
