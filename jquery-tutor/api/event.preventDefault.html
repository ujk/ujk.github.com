event.preventDefault()

event.preventDefault() Dönen Değer : undefined
Açıklama : Eğer bu metod çağrılırsa olayın default davranışı engellenir.

event.preventDefault()
Bu metod herhangi bir parametre almaz.

Örneğin tıklanan linkler tarayıcının hedef sayfaya geçmesini sağlayamaz. Mevzu olan olay için bu metodun çağrılmış olup olmadığını event.isDefaultPrevented() metodu ile kontrol edebiliriz.



Örnekler:

Tıklamanın default aksiyonunu engelle.

<a href="https://jquery.com">default click aksiyonu engellenmiş</a>
<div id="log"></div>
<script>
   $( "a" ).on( "click", function( event ) {
      event.preventDefault();
      $( "<div>" )
         .append( "default " + event.type + " önlenmiş" )
         .appendTo( "#log" );
   });
</script>
