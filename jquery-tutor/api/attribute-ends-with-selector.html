Öznitelik Şunla Biter Seçici [name$=”value”]

Öznitelik Şunla Biter Seçici
Açıklama : Öznitelik değerinin verilen test string ile bittiği elemanları seçer. Bu seçici büyük-küçük harf duyarlıdır.

jQuery( "[attribute$='value']" ) Eklendiği Versiyon 1.0
attribute : Bir bağlı özellik adı.

value : Bağlı özelliğin değeri. Tırnak içinde bir string ya da geçerli bir tanımlayıcı olabilir.

Örnekler:

Tüm input elemanları içinde name özniteliği letter ile bitenleri bul ve içeriğine bir şeyler yaz.

<input name="newsletter">
<input name="milkman">
<input name="jobletter">
 
<script>
   $( "input[name$='letter']" ).val( "a letter" );
</script>
