Class Seçicisi (“.class”)

child selector
Açıklama : Verilen class değerindeki elemanları seçer.

jQuery( ".class" ) Eklendiği Versiyon 1.0
class : Aramak için class adı. Bir eleman birçok class sahibi olabilir, sadece birinin eşleşmesi yeterlidir.

Eğer tarayıcı destekliyorsa JQuery class seçici için JavaScript getElementsByClassName() metodunu kullanır.



Örnekler:

Class değerinde "myClass" olan elemanları bul.

<style>
   div, span {
      width: 150px; height: 100px;
      float: left; padding: 10px;
      margin: 10px;
      background-color: #EEEEEE;
   }
</style>

<div class="notMe">div class="notMe"</div>
<div class="myClass">div class="myClass"</div>
<span class="myClass xc">span class="myClass xc"</span>
 
<script>
   $( ".myClass" ).css( "border", "3px solid red" );
</script>
div class="notMe"
div class="myClass"
span class="myClass xc"

Hem "myclass" hem de "otherclass" sınıf adlarının ikisine de sahip elemanları bul.

<style>
   div, span {
      width: 150px; height: 100px;
      float: left; padding: 10px;
      margin: 10px;
      background-color: #EEEEEE;
   }
</style>

<div class="myClass">div class="notMe"</div>
<div class="myClass otherclass">div class="myClass"</div>
<span class="myClass otherclass">span class="myClass"</span>

<script>
   $( ".myClass.otherclass" ).css( "border", "13px solid red" );
</script>
div class="notMe"
div class="myClass"
span class="myClass"
