JQuery API

JQuery hızlı, küçük ve zengin özelliklere sahip bir JavaScript kütüphanesidir. HTML döküman geçişleri ve manipülasyonları, olay işleme, animasyon ve AJAX gibi işleri hem kolaylaştırır hem de tarayıcıdan bağımsız çalışmasını sağlar. JQuery konusunda yeniyseniz JQuery Öğrenimi sayfasından başlayabilirsiniz.

Bu bölümde JQuery'nin çekirdek API dökümanını işleyeceğiz.



.add()
Seçilmiş olan elemanlara yeni bir eleman seçimi ekleyip, toplam elemanlardan yeni bir set oluşturur.
.addBack()
Yığına atılmış eleman setlerinden bir öncekine döner ve verilmiş olan seti ona ekleyerek yeni set oluşturur.
.addClass()
Seçilen eleman(lar)a belirtilen sınıf(lar)ı ekler.
.after()
Parametrede verilen içeriği seçilen eleman(lar)ın her birin ekler.
ajaxComplete olayı
Bir AJAX isteği tamamlandığında çalışacak bir fonksiyon iliştirilen olay. Bu bir AjaxRequest olayıdır.
