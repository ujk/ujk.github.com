:contains() Seçicisi

contains selector
Açıklama : Belirtilen yazıyı içeren elemanları seçer.

jQuery( ":contains(text)" ) Eklendiği Versiyon 1.1.4
text : Aranacak yazıyı içeren string değer. Büyük-küçük harf duyarlıdır.

Aranılan yazı direk olarak seçilen elemanın içeriğinde olabileceği gibi onun alt elemanlarında da olabilir. Öznitelik değer seçicileri gibi :contains() seçicisi parantez içinde verilen string de tırnak içinde olmalıdır.



Örnekler:

İçerik yazısında "John" olan div elemanları bulur ve altlarını çizer.

<div>John Resig</div>
<div>George Martin</div>
<div>Malcom John Sinclair</div>
<div>J. Ohn</div>
 
<script>
   $( "div:contains('John')" ).css( "text-decoration", "underline" );
</script>
John Resig
George Martin
Malcom John Sinclair
J. Ohn
