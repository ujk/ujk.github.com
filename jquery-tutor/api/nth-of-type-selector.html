:nth-of-type() Seçicisi

nth-of-type selector
Açıklama : Aynı tag olan kardeşler içinde n'incileri seçer.

jQuery( ":nth-of-type(index/even/odd/equation)" ) Eklendiği Versiyon 1.9
index : Eşleşecek çocuğun index'i, 1'den başlayan sayma yapılır. even ve odd stringi de parametre verilebilir ya da bir eşitlik (mesela :nth-of-type(even) veya :nth-of-type(4n) ).

JQuery :nth- seçicilerini CSS seçicilerden uyarladığı için index değeri 1'den başlar. .first() veya .eq() gibi metodlar JavaScript kullandığı için sıfır temelli index değerleri kullanırlar.

Ayrıntılı bilgi W3C CSS specification dökümanına bakınız.



Örnekler:

Kardeş span elemanlar içinde ikincileri bul.

<style>
   .nth { color: red; }
</style>
<div>
   <span>John</span>,
   <b>Kim</b>,
   <span>Adam</span>,
   <b>Rafael</b>,
   <span>Oleg</span>
</div>
<div>
   <b>Dave</b>,
   <span>Ann</span>
</div>
<div>
   <i><span>Maurice</span></i>,
   <span>Richard</span>,
   <span>Ralph</span>,
   <span>Jason</span>
</div>
 
<script>
   $("span:nth-of-type(2)")
      .append("<span> is 2nd sibling span</span>")
      .addClass("nth");
</script>
