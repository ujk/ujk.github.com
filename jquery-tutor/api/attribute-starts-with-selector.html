Öznitelik Şununla Başlar Seçici [name^=”value”]

Öznitelik Şununla Başlar Seçici
Açıklama : Verilen öznitelik değerinin verilen test string ile başladığı elemanları seçer.

jQuery( "[attribute^='value']" ) Eklendiği Versiyon 1.0
attribute : Bir bağlı özellik adı.

value : Bağlı özelliğin değeri. Tırnak içinde bir string ya da geçerli bir tanımlayıcı olabilir.

Bu seçici özellikle server tarafı kodlar tarafından üretilen HTML elemanlarında çok kullanılır. Elemanların ID değerleri genelde sıralı falan olduğu için aynı değerlerle başlar. Performans istiyorsanız class değerleri üzerine gitmeniz tavsiye edilir.


Örnekler:

Input boxlar içinde name değeri news ile başlayan elemanları bulur ve içlerine yazı ekler.

<input name="newsletter">
<input name="milkman">
<input name="newsboy">
 
<script>
   $( "input[name^='news']" ).val( "news here!" );
</script>
