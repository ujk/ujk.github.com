jQuery.isPlainObject()

jQuery.isPlainObject( nesne ) Dönen Değer : Boolean
Açıklama : Bir nesnenin düz nesne olmasını test eder ( "{ }"" ya da "new Object" kullanarak üretilmiş).

jQuery.isPlainObject( nesne )
nesne
Tip : Düz Nesne
Verilen nesnenin bir "düz nesne" olması test edilir.

Not : Bazı tarayıcılar için .isPlainObject() metodu hata yapabilir. Aşağıda document.location nesnesinin test edilmesi görülüyor

console.log( $.isPlainObject( document.location ) );
bu kod normalde false değer yazmalıdır. IE8 tarayıcıda hata vermektedir.



Örnekler :

Nesnelerin düz nesne olmasını test edin

jQuery.isPlainObject({}) // true
jQuery.isPlainObject( "test" ) // false
