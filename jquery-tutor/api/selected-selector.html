:selected Seçicisi

selected selector
Açıklama : Seçilmiş olan tüm elemanları bulur getirir.

jQuery( ":selected" ) Eklendiği Versiyon 1.0

:selected seçicisi <option> elemanlarında çalışır. Checkbox ve radio inputlarda çalışmaz, onlar için :checked kullanınız.


İlave Notlar:

  • :selected seçicisi bir jQuery seçicisi olup orjinal CSS seçicisi olmadığından :selected ile yapılan seçimler standart DOM querySelectorAll() metodu kadar performanslı olmaz. Bu yüzden önce elemanları saf CSS seçiciler ile seçip sonra .filter(":selected") kullanarak :selected seçicisi kullanınız.
