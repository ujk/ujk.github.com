Descendant (torun) Seçici (“ancestor descendant”)

descendant selector
Açıklama : Verilen ata eleman (ancestor) içinde descendant ile belirtilen torunları (alt elemanları) bulur.

jQuery( "ancestor descendant" ) Eklendiği Versiyon 1.0
ancestor : İçinde torunlar aranacak ata elemanı (üst eleman) seçen geçerli bir seçici deyim.

descendant : Aranacak torunları (alt elemanları) bulan seçici deyim.

Bir elemanın torunu onun çocuğu, torunu, torununun çocuğu gibi içinde yer alan tüm elemanlardır. İçiçe elemanların en içindekine kadar sorgular.



Örnekler:

Form içindeki tüm input elemanı torunları bul ve işaretle. Formun torunu fieldset elemanı torunu olanlara da sarı arka renk ver.

<style>
   form {
      border: 2px green solid;
      padding: 2px; margin: 0;
      background: #efe;
   }
   div { color: red; }
   fieldset {
      margin: 1px; padding: 3px;
   }
</style>
<form>
   <div>Form yeşil sınırla çizilmiştir.</div>
   
   <label for="name">Formun çocuğu:</label>
   <input name="name" id="name">
   
   <fieldset>
      <label for="newsletter">Formun torunu, fieldsetin çocuğu:</label>
      <input name="newsletter" id="newsletter">
   </fieldset>
</form>
Formun kardeşi: <input name="none">
 
<script>
   $( "form input" ).css( "border", "2px dotted blue" );
   $( "form fieldset input" ).css( "backgroundColor", "yellow" );
</script>
Form yeşil sınırla çizilmiştir.
