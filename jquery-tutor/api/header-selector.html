:header Seçicisi

header selector
Açıklama : Header (h1, h2, h3, h4, h5, h6) elemanları seçer.

jQuery( ":header" ) Eklendiği Versiyon 1.2

İlave Notlar :

  • Bu :header seçicisi bir JQuery eklentisi olup standart CSS seçici olmadığı için, standart DOM querySelectorAll() metodunun performansında olamaz. Modern tarayıcılarda daha iyi performans için önce grup seçimini standart CSS ile yapıp .filter(":header"). metodu ile filtreleyiniz.


Örnekler:

Sayfadaki tüm başlıklara arka renk ve yazı rengi ver.

<h1>Header 1</h1>
<p>Contents 1</p>
<h2>Header 2</h2>
<p>Contents 2</p>
 
<script>
   $(":header").css({ background: "#ccc", color: "blue" });
</script>
