:only-of-type Seçicisi

only-of-type selector
Açıklama : Kardeşleri içinde aynı tag tipinde olmayan tekleri seçer.

jQuery( ":only-of-type" ) Eklendiği Versiyon 1.9

Eğer ebeveynin aynı eleman adında başka çocuğu varsa hiçbir şey ile eşleşmez.



Örnekler:

Ebeveyninin tek button elemanı çocuğu olan butonların yazılarıyla oyna.

<style>
   div {
      width: 100px; height: 80px;
      margin: 5px; float: left;
      background: #b9e;
   }
   span {
      padding: 2px; margin: 3px;
      line-height: 1.4;
      border: 1px solid #000;
   }
</style>
<div>
   <button>Sibling!</button>
   <button>Sibling!</button>
</div>
<div>
   <button>Sibling!</button>
</div>
<div>
   None
</div>
<div>
   <button>Sibling!</button>
   <span>Sibling!</span>
   <span>Sibling!</span>
</div>
<div>
   <button>Sibling!</button>
</div>
 
<script>
   $("button:only-of-type").text("Alone")
      .css("border", "2px blue solid");
</script>
None
Sibling! Sibling!
