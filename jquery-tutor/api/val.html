.val() Metodu

Seçilen elemanlar setinin ilk elemanının değerini oku ya da elemanların hepsinin değerini değiştir.

İçindekiler :

  • .val()
    • .val()
  • .val( value )
    • .val( value )
    • .val( function )
.val() Dönen Değer : String veya Sayı veya Array
Açıklama : Seçilen elemanların ilkinin değerini okur.
.val() Eklendiği Versiyon 1.0
Bu şekil kullanım bir parametre almaz.

Bu şekil .val() metodu kullanımı örn. <input>, <select> ve <textarea> gibi elemanlardan değer okumak için kullanılır. Boş bir eleman setinde kullanılınca undefined döner.

Kolleksiyondaki ilk eleman çoklu-seçim bir elemansa (mesela multiple özelliği set edilmiş bir select elemanı) .val() metodu elemanlarında seçilen değerleri içeren bir array döner. JQuery 3.0 ile eğer seçili opsiyon yoksa boş array döner, jQuery 3.0 öncesi ise null değer döner.

Select, checkbox ve raido buttonlar için doğru elemanları seçmek için checked özelliğini kullanabilirsiniz.
