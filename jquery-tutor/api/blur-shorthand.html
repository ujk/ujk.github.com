Bu şekilde kullanım herhangi bir parametre almaz.

Bu API kaldırıldı

.blur( handler ) veya .blur( eventData, handler ) yerine sırasıyla
.on( "blur", handler ) ve .on( "blur", eventData, handler ) kullanınız.

.blur() yerineyse .trigger( "blur" ) kullanınız.
