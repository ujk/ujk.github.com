:first Seçicisi

first selector Vazgeçilen versiyon 3.4
Açıklama : Eşleşen setteki ilk DOM elemanı seçer.

jQuery( ":first" ) Eklendiği Versiyon 1.0

JQuery 3.4 ile :first sahte seçicisinden vazgeçildi. Eğer varsa kodunuzdan çıkarıp yerine .first() metodunu kullanınız.

:first sahte sınıfı aslında :eq(0) ile aynı işi yapar. Ayrıca :lt(1) olarak da yazılabilir. Bu tek bir eleman seçerken , :first-child seçicisi her ebeveyn için bir eleman döner.



İlave Notlaar :

  • :first standart CSS olmayıp bir jQuery eklentisi olduğu için JavaScript querySelectorAll() metodu performansında olamaz. Daha iyi performans için :first kullanmadan önce standart bir CSS seçici ile grup seçiniz ve sonrasında .filter(":first") ile sonucu filtreleyiniz.
  • Elemanlar döküman içinde geldiği sırayla dizilir.


Örnekler:

İlk tablo satırını bul.

<style>
   td {
      color: blue;
      font-weight: bold;
   }
</style>
<table>
   <tr><td>Row 1</td></tr>
   <tr><td>Row 2</td></tr>
   <tr><td>Row 3</td></tr>
</table>
<script>
   $( "tr:first" ).css( "font-style", "italic" );
</script>
Row 1
Row 2
Row 3
