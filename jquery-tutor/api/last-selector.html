:last Seçicisi

last selector Vazgeçilen versiyon 3.4
Açıklama : Eşleşen setteki son DOM elemanı seçer.

jQuery( ":last" ) Eklendiği Versiyon 1.0

JQuery 3.4 ile :last sahte seçicisinden vazgeçildi. Eğer varsa kodunuzdan çıkarıp yerine .last() metodunu kullanınız.

:last seçicisi verilen eleman setinin en son elemanını içeren bir jQuery nesnesi döner.



İlave Notlaar :

  • :last standart CSS olmayıp bir jQuery eklentisi olduğu için JavaScript querySelectorAll() metodu performansında olamaz. Daha iyi performans için :last kullanmadan önce standart bir CSS seçici ile grup seçiniz ve sonrasında .filter(":last") ile sonucu filtreleyiniz.


Örnekler:

Son tablo satırını bul.

<table>
   <tr><td>First Row</td></tr>
   <tr><td>Middle Row</td></tr>
   <tr><td>Last Row</td></tr>
</table>
 
<script>
   $("tr").last().css({
      backgroundColor: "yellow",
      fontWeight: "bolder" });
</script>
