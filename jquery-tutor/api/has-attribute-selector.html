Öznitelik Seçici [name]

attributeHas selector
Açıklama : Herhangi bir değerde verilen öznitelik var olanları bulur.

jQuery( "[attribute]" ) Eklendiği Versiyon 1.0
attribute : Bir öznitelik (bağlı özellik) adı.


Örnekler:

ID değeri olan <div> elemanlarına tıklandığında ID değerini ekleyen bir olay işleyici bağla.

<div>ID yok</div>
<div id="hey">ID var</div>
<div id="there">bunda da ID var</div>
<div>bişey yok</div>
 
<script>
   // .one() kullanarak olay işleyicinin sadece bir
   // kere çalışması sağlanır
   $( "div[id]" ).one( "click", function() {
      var idString = $( this ).text() + " = " + $( this ).attr( "id" );
      $( this ).text( idString );
   });
</script>
ID yok
ID var
bunda da ID var
bişey yok
