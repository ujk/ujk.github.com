:last-of-type Seçicisi

last-of-type selector
Açıklama : Aynı tag olan kardeşler içinde sonuncuyu seçer.

jQuery( ":last-of-type" ) Eklendiği Versiyon 1.9

:last-of-type seçicisi aynı ebeveyn altında bulunan aynı tag adına sahip elemanların döküman ağacında son sırada olanını seçer.



Örnekler:

Eşleşen her <div> elamanı içindeki son <span> elemanını bulup ona biraz css ve hover etkisi ver.

<style>
   span.solast {
      text-decoration: line-through;
   }
</style>
<div>
   <span>Corey,</span><span>Yehuda,</span>
   <span>Adam,</span><span>Todd</span>
</div>
<div>
   <span>Jörn,</span><span>Scott,</span>
   <span>Timo,</span><b>Nobody</b>
</div>
<script>
$("span:last-of-type")
   .css({color:"red", fontSize:"80%"})
   .hover(function() {
      $(this).addClass("solast");
   }, function() {
      $(this).removeClass("solast");
   });
</script>
Corey,Yehuda, Adam,Todd
Jörn,Scott, Timo,Nobody
