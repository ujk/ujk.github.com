Çoklu Öznitelik Seçici [name=”value”][name2=”value2”]

attributeMultiple selector
Açıklama : Tüm Öznitelik filtrelerinin geçerli olduğu elemanları seçer.

jQuery( "[attributeFilter1][attributeFilter2][attributeFilterN]" ) Eklendiği Versiyon 1.0
attributeFilter1 : Bir bağlı özellik seeçici.

attributeFilter2 : Bir diğer bağlı özellik seeçici.

attributeFilterN : Ne kadar öznitelik seçici gerekirse eklenir.


Örnekler:

Tüm id özniteliği olan ve name özniteliği de "man" ile biten <input> elemanları bulur ve içine yazı yazar.

<input id="man-news" name="man-news">
<input name="milkman">
<input id="letterman" name="new-letterman">
<input name="newmilk">
 
<script>
   $("input[id][name$='man']").val("sadece bu!");
</script>
