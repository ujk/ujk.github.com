Sonraki Bitişik Seçicisi (“prev + next”)

next adjacent selector
Açıklama : "prev" ile verilenin hemen sonrasında bitişik gelen "next" ile belirtilen kardeş eleman(lar)ı seçer.

jQuery( "prev + next" ) Eklendiği Versiyon 1.0
prev : Herhangi bir seçici.

next : İlk seçiciden sonra gelen elemanı seçmek için bir seçici.

Sonraki bitişik seçicisi prev + next ve herhangi bir kardeş eleman seçicisi prev ~ siblings 'in en önemli ayrıntısı verilen iki parametredeki elemanlar aynı ebeveyn içinde olmalıdır.



Örnekler:

label sonrasında gelen tüm input elemanları bul.

<form>
   <label for="name">Name:</label>
   <input name="name" id="name">
   <fieldset style="border: 1px solid; padding: 3px;">
      <label for="newsletter">Newsletter:</label>
      <input name="newsletter" id="newsletter">
   </fieldset>
</form>
<input name="none">
 
<script>
   $("label + input").css("color", "blue").val("Labeled!");
</script>
