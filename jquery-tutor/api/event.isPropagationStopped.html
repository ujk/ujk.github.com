event.isPropagationStopped()

event.isPropagationStopped() Dönen Değer : Boolean
Açıklama : Bu olay nesnesinde event.stopPropagation() çağrılmış mı test eder.

event.isPropagationStopped()
Bu metod bir parametre almaz

Bu olay metodu W3C DOM Level 3 specification makalesinde anlatılmıştır.




Örnekler:

event.stopPropagation() metodu çağrılmış mı kontrol eder.

<button>bana tıkla</button>
<div id="stop-log"></div>
 
<script>
function propStopped( event ) {
   var msg = "";
   if ( event.isPropagationStopped() ) {
      msg = "çağrıldı";
   } else {
      msg = "çağrılmadı";
   }
   $( "#stop-log" ).append( "<div>" + msg + "</div>" );
}
 
$( "button" ).on( "click", function( event ) {
   propStopped( event );
   event.stopPropagation();
   propStopped( event );
} );
</script>
