Bağlı Özellik Kelime Seçici [name~=”value”]

Öznitelik Kelime Seçici
Açıklama : Öznitelik (attribute) değerinde verilen string boşlukla ayrılmış ayrı kelime olarak bulunanları seçer.

jQuery( "[attribute~='value']" ) Eklendiği Versiyon 1.0
attribute : Bir bağlı özellik adı.

value : Bağlı özelliğin değeri. Tırnak içinde bir string ya da geçerli bir tanımlayıcı olabilir.

Bu seçici belirtilen öznitelik değerindeki kelimelerde belirtilen string değeri arar. Kelime string içinde boşlukla ayrılmış durumdadır. Seçici verilen test kelimesinin tam eşleştiği elemanları döner.


Örnekler:

Tüm input elemanları içinde name özniteliğinde man kelimesi olanları bul ve içeriğine bir şeyler yaz.

<input name="man-news">
<input name="milk man">
<input name="letterman2">
<input name="newmilk">
 
<script>
   $( "input[name~='man']" ).val( "mr. man is in it!" );
</script>
