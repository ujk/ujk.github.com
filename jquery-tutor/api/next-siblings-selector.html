Sonraki Kardeşler Seçicisi (“prev ~ next”)

next siblings selector
Açıklama : "prev" ile verilenin arkasından gelen tün belirtilen kardeş elemanları seçer.

jQuery( "prev ~ next" ) Eklendiği Versiyon 1.0
prev : Herhangi bir seçici.

next : İlk seçiciden sonra gelen kardeşleri seçmek için bir seçici.

Sonraki bitişik seçicisi prev + next ve sonraki kardeş elemanlar seçicisi prev ~ siblings 'in en önemli ayrıntısı verilen iki parametredeki elemanlar aynı ebeveyn içinde olmalıdır.



Örnekler:

ID değeri #prev olan elemandan sonra gelen tüm <div> kardeşleri seçer.

<style>
   div, span {
      display: block; width: 80px;
      height: 80px; margin: 5px;
      background: #bfa;
      float: left; font-size: 14px;
   }
   div#small {
      width: 60px; height: 25px;
      font-size: 12px; background: #fab;
   }
</style>
<div>div (#prev'den önce - seçilmez)</div>
<span id="prev">span#prev</span>
<div>div kardeş </div>
<div>div kardeş <div id="small">div yeğen</div></div>
<span>span kardeş (div değil)</span>
<div>div kardeş</div>
 
<script>
   $("#prev ~ div").css("border", "3px groove blue");
</script>
